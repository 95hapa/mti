Install instructions for the different Ubuntu versions.

Installing each VM (Ubuntu 12.04.5, Ubuntu 14.04.5, and Ubuntu 16.04.1) on Hyper-V:
1) Seting up each VM on Hyper-V:
	a) 8 GB of RAM.
	b) 50 GB of HDD (dynamic virtual hard disc).
2) When installing the OS on each respective VM:
	a) Wipe hard-drive and install the OS.
	b) Do NOT use LVM for easy-partitioning (just wipe the drives and restart again if you messed up).

Password for all Ubuntu VMs: passwordMTI

Download the source of charade GRAMMy using git:
git clone https://bitbucket.org/charade/grammy.git

For all Ubuntu versions, set the environment with the following terminal commands:
sudo apt-get update && sudo apt-get upgrade
sudo apt-get install build-essential libstdc++6 python2.7 python-numpy python-scipy python-setuptools python-dev python-biopython r-base r-base-dev
sudo easy_install rpy2

After setting up the environment, put the charade GRAMMy source files into the VMs,
cd into the GRAMMy folder, and fo the following terminal commands:
sudo python setup.py install
cd test
. test.sh
